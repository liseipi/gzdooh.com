华中电网公司智能电网调度技术支持系统项目取得突破性进展
智能电网调度技术支持系统项目取得突破性进展，再过两个月该项目将进入推广测试阶段。这是记者2月23日从智能电网调度技术支持系统华中试点项目会战现场动员会上获悉的消息。华中电网公司副总经理陈晓伦，国家电网公司调度中心副主任辛耀中出席会议并作讲话，中国电力科学院，国家电网科学院的有关领导及项目组相关人员参加会议。 　　据了解，在国家电网公司制订的智能电网调度技术支持系统建设框架中，华中电网公司承担试点的建设内容主要侧重于基础平台及实时监控与预警基础应用功能，这也是智能电网调度技术支持系统的核心和根本所在。目前，工程的主要开发工作已经基本完成，正在进行紧张的测试及系统消缺，4月份即将进入到安装推广试用。 　　陈晓伦感谢国家电网调度中心及中国电力科学院和国家电网科学院对华中公司此试点项目的大力支持，他表示，华中电网公司非常重视这个试点项目，投入了大量的人力物力，及时了解项目进展，特事特办对此项目进行配合与支持，目前此项目取得了突破性进展，华中公司将全力以赴做好这项工作，保证质量，确保项目按期完成。 　　辛耀中对华中电网公司的大力支持表示感谢，他希望中国电力科学院、国家电网科学院和华中电网公司通力合作，进一步增加技术力量，消除系统测试过程中发现的问题，早日完成收官任务。同时他对项目的下一步工作作出了具体部署。 来源：国际电力网
